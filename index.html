Search Bar with Hyperlink Button

This example shows a search bar with a button that contains a hyperlink. The button is styled with a gradient background.

Search 
<!-- HTML Code -->
<div class="search-bar">
    <input type="text" placeholder="Search for tools or resources...">
    <a href="https://www.example.com/search" class="search-button">
        <i class="fas fa-search"></i> Search
    </a>
</div>

<!-- CSS Code -->

    position: relative;

    transition: var(--transition);

    position: absolute;
    right: 8px;
    top: 8px;

    color: white;
    border: none;
    border-radius: 50px;
    padding: 10px 25px;
    cursor: pointer;
    transition: var(--transition);
    display: flex;
    align-items: center;
    gap: 8px;
    text-decoration: none;
}

.search-bar .search-button:hover {
    transform: translateY(-2px);
    box-shadow: 0 5px 15px rgba(0, 0, 0, 0.2);
    background: linear-gradient(135deg, var(--secondary) 0%, var(--primary) 100%);
}

Implementation Note

To make the button a hyperlink, replace the <button> element with an <a> tag while keeping the same CSS styles.

Responsive Design

On mobile devices, the button moves below the input field for better usability and touch interaction.
